Building Tramonto

Installing Tramonto is almost too easy. In order to install Tramonto on this virtual machine you must first download the source code from the Tramonto web page. After downloading the source code create a separate build directory. Create a cmake configure script using the examples contained in the source as a template. Within the configure script instead of using the -DTRILINOS_DIR flag use the -DCMAKE_PREFIX_PATH flag to point to the trilinos installation. In the end the contents will look similar to this:

cmake \
-DCMAKE_PREFIX_PATH:PATH=/usr/local/ACTS/gnu-4.6.2/openmpi-1.5.4/trilinos-10.8.5/ \
../Tramonto-4.0.1-Source

Then simply run the configure script to correctly configure everything and finally type make to build Tramonto. In order to run examples using Tramonto move to the desired example directory then run the example by giving the path to the executable dft_exec from the desired example. For more information on using Tramonto see the Tramonto Website or the READ_ME files in the source directory.
